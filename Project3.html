"That's what I said. Sodium Chloride" -Jimmy Neutron


For Project 3 we I came up with a small enclosure for my device that utilizes 1 potentiometer (with a sick knob), a momentary button, a toggle button, a <1 Ohm speaker, a green LED, a UV LED, and a photovoltaic sensor. The video demonstrates ther functionality of the device, and I knew I wanted to make it smaller to be able to hold it in my hand better. Plus small little technology or devices are always intriguing.



This was my design. The knobs and buttons were put in places that would allow for the best and easist interface for users. The right thumb can hold the button producing sound with the left hand is free to move the potentiometer and for covering the photo cell also. All of the equipment was connected to an Arduino Uno microchip with jumper wires and a breadboard.

VIDEO DOCUMENTATION HERE

When the toggle switch is in the on position, if the user presses the momentary switch the speaker plays a note. The pitch of this note is determined by the position of the potentiometer with the farthest couter-clockwish extreme representing the lowest tone and the farthest clockwish extreme being the highest tone. While the toggle switch is in the on position, the green LED remains in the on state, and the UV LED brightness is mapped to the value of the potentiometer so that when it is turned all the way down and the tone is low, the LED is dim. When the tone is high and the potentiometer is turned all the way up, the LED is the brightest.

#include "pitches.h"
int notes = 52;
int scale[] = {
NOTE_B0, NOTE_C1, NOTE_D1, NOTE_E1, NOTE_F1, NOTE_G1, NOTE_A1, NOTE_B1, NOTE_C2,
NOTE_D2, NOTE_E2, NOTE_F2, NOTE_G2, NOTE_A2,
NOTE_B2, NOTE_C3, NOTE_D3, NOTE_E3, NOTE_F3, NOTE_G3, NOTE_A3, NOTE_B3, NOTE_C4,
NOTE_D4, NOTE_E4, NOTE_F4, NOTE_G4, NOTE_A4,
NOTE_B4, NOTE_C5, NOTE_D5, NOTE_E5, NOTE_F5, NOTE_G5, NOTE_A5, NOTE_B5, NOTE_C6,

NOTE_B6, NOTE_C7, NOTE_D7, NOTE_E7, NOTE_F7, NOTE_G7, NOTE_A7, NOTE_B7, NOTE_C8, NOTE_D8
};
int scaleDurations[] = {
8, 8, 8, 8, 8, 8, 8, 8, 8, 8, 8, 8, 8, 8, 8, 8, 8, 8, 8, 8, 8, 8, 8, 8, 8, 8, 8, 8, 8,
8, 8, 8, 8, 8, 8, 8, 8, 8, 8, 8, 8, 8, 8, 8, 8, 8, 8, 8, 8, 8, 8, 8
};
const int speakPin = 9; // the number of the speaker pin
const int button1Pin = 4; // momentary
const int button2Pin = 5; // toggle
const int led1Pin = 10; // LED digital out
const int led2Pin = 11; // Analog digital out
const int analogInPin0 = A0; //analog input pin for pot
const int analogInPin1 = A1; //analog input pin for photcell
int sensorValue0 = 0; // value read from the pot
int outputValue0 = 0; // value output for map
int sensorValue1 = 0; // value read from the photcell
int outputValue1 = 0; // value output for map
int button1State = 0; // variable for reading the momentary button status
int button2State = 0; //variable for togle button
int led1Value = 0; //variable for brightness of LED 1
int led2Value = 0; //variable for brightness of LED 2
void setup() {
Serial.begin(9600);
pinMode(led1Pin, OUTPUT);
pinMode(led2Pin, OUTPUT);
pinMode(button1Pin, INPUT);
pinMode(button2Pin, INPUT);
pinMode(analogInPin0, INPUT);
pinMode(analogInPin1, INPUT);
}
void loop() {
button1State = digitalRead(button1Pin);
button2State = digitalRead(button2Pin);
sensorValue0 = analogRead(analogInPin0);
sensorValue1 = analogRead(analogInPin1);
outputValue0 = map(sensorValue0, 0, 1023, 0, 51);
led1Value = map(sensorValue0, 0, 1023, 0, 255);
led2Value = map(sensorValue1, 0, 1023, 255, 0);
if (button2State == HIGH){
analogWrite(led2Pin, led2Value);
if (button1State == HIGH) {
int noteDuration = 1000 / scaleDurations[outputValue0];
tone(speakPin, scale[outputValue0], noteDuration);
analogWrite(led1Pin, led1Value);
} else {
noTone(speakPin);
analogWrite(led1Pin, 0);
}
} else {
analogWrite(led2Pin, 0);
analogWrite(led1Pin, 0);
}
}
